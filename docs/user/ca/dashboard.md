# Dashboard

## Que fa
Mostra el resum mensual de la llar actual i un acces rapid per sortir.

## Com s'utilitza
1. Iniciar sessio.
2. Veure el resum mensual de la llar seleccionada.
3. Usar "Tancar sessio" per sortir.

## Exemple
- Ingressos, despeses i balanc del mes actual.

## Casos limit
- Si no hi ha llar seleccionada, es mostra un avis.
