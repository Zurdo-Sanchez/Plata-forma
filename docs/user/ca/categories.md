# Categories

## Que fa
Classifica moviments per a reportes i filtres.

## Com s'utilitza
1. Seleccionar una llar.
2. Crear categories amb nom.
3. Usar categories a les linies de transaccio.
4. Editar categories quan calgui.
5. Eliminar categories que no s'usen (borrat suau).

## Exemple
- Categoria "Salari".
- Categoria "Supermercat".

## Casos limit
- El borrat es suau: la categoria queda inactiva i no apareix en llistes noves.
- Les categories inactives no s'han d'usar en noves transaccions.
- Si no hi ha categories, el reporte agrupa com "Sense categoria".
