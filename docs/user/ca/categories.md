# Categories

## Que fa
Classifica ingressos, despeses i transferencies per a reportes i filtres.

## Com s'utilitza
1. Seleccionar una llar.
2. Crear categories amb nom i tipus.
3. Usar categories a les linies de transaccio.
4. Arxivar categories que no s'usen.

## Exemple
- Categoria "Salari" tipus INCOME.
- Categoria "Supermercat" tipus EXPENSE.

## Casos limit
- Les categories arxivades no s'han d'usar en noves transaccions.
- Si no hi ha categories, el reporte agrupa com "Sense categoria".
