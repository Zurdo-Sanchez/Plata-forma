# Hogares

## Que hace
Permite crear hogares, seleccionar el hogar activo y administrar miembros.

## Como se usa
1. Ir a la pantalla Hogares.
2. Crear un hogar con nombre y moneda opcional.
3. Seleccionar el hogar actual desde la lista.
4. Agregar miembros por email (solo OWNER).

## Ejemplo
- Crear "Casa" con moneda "USD".
- Agregar miembro "user@example.com" con rol MEMBER.

## Casos limite
- Solo OWNER puede editar o agregar miembros.
- Si no hay hogar seleccionado, otras pantallas muestran un aviso.
