# Dashboard

## Que hace
Muestra el resumen mensual del hogar actual y un acceso rapido para salir.

## Como se usa
1. Iniciar sesion.
2. Ver el resumen mensual del hogar seleccionado.
3. Usar "Cerrar sesion" para salir.

## Ejemplo
- Ingresos, gastos y balance del mes actual.

## Casos limite
- Si no hay hogar seleccionado, se muestra un aviso.
