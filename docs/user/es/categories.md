# Categorias

## Que hace
Clasifica movimientos para reportes y filtros.

## Como se usa
1. Seleccionar un hogar.
2. Crear categorias con nombre.
3. Usar categorias en las lineas de transaccion.
4. Editar categorias cuando sea necesario.
5. Borrar categorias que no se usan (borrado suave).

## Ejemplo
- Categoria "Salario".
- Categoria "Supermercado".

## Casos limite
- El borrado es suave: la categoria queda inactiva y no aparece en listas nuevas.
- Las categorias inactivas no deben usarse en nuevas transacciones.
- Si no hay categorias, el reporte agrupa como "Sin categoria".
