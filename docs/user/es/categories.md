# Categorias

## Que hace
Clasifica ingresos, gastos y transferencias para reportes y filtros.

## Como se usa
1. Seleccionar un hogar.
2. Crear categorias con nombre y tipo.
3. Usar categorias en las lineas de transaccion.
4. Archivar categorias que no se usan.

## Ejemplo
- Categoria "Salario" tipo INCOME.
- Categoria "Supermercado" tipo EXPENSE.

## Casos limite
- Las categorias archivadas no deben usarse en nuevas transacciones.
- Si no hay categorias, el reporte agrupa como "Sin categoria".
