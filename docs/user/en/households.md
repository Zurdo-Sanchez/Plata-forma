# Households

## What it does
Lets you create households, select the active household, and manage members.

## How to use
1. Open the Households screen.
2. Create a household with name and optional currency.
3. Select the current household from the list.
4. Edit name or currency when needed.
5. Delete a household if it is no longer used.
6. Add members by email (OWNER only).

## Example
- Create "Home" with currency "USD".
- Add member "user@example.com" with role MEMBER.

## Edge cases
- Only OWNER can edit or add members.
- Deleting a household removes its accounts, categories, transactions, cards, loans, and reports.
- If no household is selected, other screens show a warning.
