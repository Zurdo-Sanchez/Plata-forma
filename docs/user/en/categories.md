# Categories

## What it does
Classifies movements for reports and filters.

## How to use
1. Select a household.
2. Create categories with a name.
3. Use categories in transaction lines.
4. Edit categories when needed.
5. Delete categories you no longer use (soft delete).

## Example
- Category "Salary".
- Category "Groceries".

## Edge cases
- Deletion is soft: the category becomes inactive and is hidden from new lists.
- Inactive categories should not be used in new transactions.
- If there are no categories, reports group as "Uncategorized".
