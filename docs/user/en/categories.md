# Categories

## What it does
Classifies income, expense, and transfer entries for reports and filters.

## How to use
1. Select a household.
2. Create categories with name and type.
3. Use categories in transaction lines.
4. Archive categories you no longer use.

## Example
- Category "Salary" type INCOME.
- Category "Groceries" type EXPENSE.

## Edge cases
- Archived categories should not be used in new transactions.
- If there are no categories, reports group as "Uncategorized".
